SatDatepicker features

Features from original datepicker

Range selection

In order to enable range selection, just pass the option selectionModes with the selection modes you want to enable on the calendar. Additionally, you can pass the option initialSelectionMode to indicate the initial selection mode.


selectionModes Array indicating the selection modes available. Possible values are: date, range, since and until. The buttons for each selection mode will be created in the order passed to the calendar, making it easy to order them.

initialSelectionMode Sets the initial selection mode. If the input already has a value, this option is ignored and the selection mode is identified by the value.

Close after selection

closeAfterSelection option enables or disables datepicker closing after date selections. By default the option is true

Calendar view order

orderPeriodLabel option sorts the label period views.
  • Default [multi-year]: multi-year then back to month
  • Month [month]: month > year > multi-year

Select first date on close

selectFirstDateOnClose option enables or disables the selection of the first date when closing the datepicker before selecting another date. By default the option is false

Custom Footer (Default Ranges)

calendarFooterComponent option allows you to define a footer in the calendar simalar to custom headers.

Inline Usage

The sat-calendar component with its dateRangesChange output binding can be used as an inline range date picker.
{{ inlineRange.begin | date }} {{ inlineRange.end | date }}
